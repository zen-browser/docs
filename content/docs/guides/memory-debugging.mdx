## ‚úÖ Troubleshooting Steps

1. Open `about:profiles`
2. Click **Create a New Profile**
3. Compare memory usage between:
   - Your **main profile**
   - The **new clean profile**
4. If the new profile does **not** show memory issues:
   - The problem may be caused by extensions, mods, themes, or user configuration
   - Try disabling or removing them and retest
5. If the new profile **still** shows high memory usage:
   - The issue may be internal to Zen
   - Collect a memory report and **open an issue** with details
